2D Maze

Витрачено часу всього (приблизно): 20 годин
- Кор-механіки (генерація, пошук шляху, плеєр) + ресьорч: 6 годин
- Інтерфейси (каунтери, вікна, взаємодія): 6 годин
- Сейви (включно зі снапшотом останньої сесії): 3 години
- Візуал + пошук ассетів + партікли: 2 години
- Полішинг, стратегії мувменту, перевірка на девайсі: 3 години

Реалізовано три алгоритма генерації лабіринту:
- пошук в глибину;
- бінарне дерево;
- алгоритм Прима.

Параметри лабіринту задаються в _MazeModel_:
- вибір алгоритму генерації;
- мінімальна та максимальна ширина;
- мінімальна та максимальна висота;
- мінімальна та максимальна кількість виходів.
Останні три параметра використовуються як ліміти для налаштувань перед грою у вікні _SettingsDialog_.

Рух гравця - просте пересування трансформу, без фізики. Для детекту стін використовується рейкаст.
Логіка інпута реалізована стратегіями - _KeyboardInputStrategy_ для редактора та _JoystickInputStrategy_ для девайсів.
